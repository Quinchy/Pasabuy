# About

### Title:
Pasabuy

### Description:
Pasabuy is a peer-to-peer delivery platform that allows users to either post pasabuy or join existing posts made by others.

# Features

• Post Items or Foods
• Account and Profile Management
• Ask for Pasabuy
• Sell and Deliver
• Manage Your Pasabuy Posts
• Messaging
• Track Orders
• Rating
